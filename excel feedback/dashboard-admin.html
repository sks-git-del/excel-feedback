ℹ️ PIN Information

PIN System Features:

  • ✓ Extra security layer for Faculty and Admin accounts
  • ✓ All admins use one PIN, all faculty use one PIN
  • ✓ Students do NOT need PIN - password only
  • ✓ PIN must be 4-6 digits (numbers only)
  • ✓ PIN is required in addition to password for Faculty & Admin
  • ✓ You (admin) can change both PINs anytime
  • ✓ PIN system can be toggled on/off at any time
